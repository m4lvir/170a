Professional Development

Advice and resources from colleagues and mentors.

Topic 1: Effective Communication in Meetings

Advice received: Always summarize the key decisions and action items at the end of a meeting, and send a quick email follow-up within the hour.

Topic 2: Best Practices for Code Review

Advice received: Focus on intent and logic first. Use automated linting for style issues so you can concentrate on the architecture and security implications.

Resources:

  • List of Recommended Linting Tools
  • Book: The Pragmatic Programmer (highly recommended for developers).
